# SnapQR2Link - 製品仕様書

## 1. プロダクト概要

### 1.1 プロダクト名
SnapQR2Link（スナップQRツーリンク）

### 1.2 プロダクトの説明
スクリーンショットなど画像内のQRコードを解析し、リンクにジャンプするWebアプリケーション

### 1.3 プロダクトの目的
画面上のQRコード（Zoom招待、チラシPDF、SNS上の画像など）をカメラでかざすことなく、簡単に読み取り・利用できるようにする

---

## 2. ターゲットユーザー

### 2.1 想定ユーザー
- スマホやPCでスクリーンショットをよく撮る人
- カメラでかざせない画面上のQRコード（Zoom招待、チラシPDF、SNS上の画像など）を簡単に開きたい人

### 2.2 ユーザーペルソナ

| （追記可能） |  |  |  |  |

### 2.3 ユーザーストーリー
- [ ] ユーザーとして、スクリーンショット内のQRコードを読み取りたい
- [ ] ユーザーとして、読み取ったURLをワンクリックで開きたい
- [ ] ユーザーとして、複数のQRコードがある場合は選択して開きたい
- [ ] ユーザーとして、URL以外のテキストもコピーできるようにしたい

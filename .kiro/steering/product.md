# Product Overview

SnapQR2Link（スナップQRツーリンク）は、スクリーンショット内のQRコードを解析し、リンクへ直接ジャンプできるWebアプリケーションです。

## Core Capabilities

- **画像内QRコード検出**: アップロードされた画像から自動的にQRコードを検出・解析
- **即座のリンクアクセス**: URLを含むQRコードをワンクリックで開く
- **複数QRコード対応**: 1枚の画像に複数のQRコードが含まれる場合も全て検出して選択可能
- **プライバシー重視**: すべての処理をクライアントサイドで完結（画像はサーバーに送信されない）

## Target Use Cases

- Zoom招待など画面上のQRコードをカメラでかざせない状況
- PDF資料やSNS投稿内のQRコードを素早く開きたい
- リモートワークでのミーティング参加をスムーズに
- スクリーンショット保存済みのQRコードを後から開く

## Value Proposition

**"画面上のQRコードを、カメラ不要で即座に開く"**

従来はスマホのカメラでPC画面をかざす必要があったQRコードを、スクリーンショットをアップロードするだけで読み取れます。プライバシーを守るため、すべての処理はブラウザ内で完結します。
